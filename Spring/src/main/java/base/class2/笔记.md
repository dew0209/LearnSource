4.给容器中注册bean的方式
    @Bean：导入第三方的类或者组件
    包扫描 + 组件的标注注解（@ComponentScan @Controoler @Service @Repository @Component）：一般是针对我们自己写的类
    @Import：快速给容器导入一个组件。
        a.@Import(要导入到容器中的组件)：容器会自动注册这个组件，bean的id为全类名
        b.ImportSelector:是一个接口，返回需要导入到容器的组件的全类名数组
        c.ImportBeanDefinitionRegistrar：可以手动添加组件到IOC容器，所有bean的注册可以BeanDefinitionRegistry
